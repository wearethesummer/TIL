# HTML5 핵심 태그

- 제목표시하기 ```h1~h6```
  
- 문단, 줄 바꿈 ```p, br```
  - ```
    <p> 태그는 문단. <p> 안에 자식 요소로 잡는다.
    <br> 태그는 줄바꿈. <p> 안에 줄을 바꾸고 싶을 때 사용한다.
    ```
  
- 글자 굵게 표시하기 ```b, strong(강조)```
  
- 글자 기울임 표시하기 ```em, I```
  
- 글자 밑줄 표시하기 ```u```
  
- 글자 취소선 표시하기 ```s, del```
  
- 큰 글자 표시, 작은 글자 표시 ```big, small```
  
- 위 첨자, 아래 첨자 ```sup, sub```
  
- 형관펜 배경색 글자 ```mark```
  
- 수평선 넣기 ```hr```

- 순서가 있는 목록 ```ol > li```

- 순서가 없는 목록 ```ul > li```

- 외부 링크나 파일 링크하기 ```a```
  <br/>
  - ``` <a href=""></a>```
- 이미지 html 문서에 넣기 ```img```
  <br/>
  - ```<img scr=""/>```

- 링크와 이미지 중첩하기 : a > img
  <br/>
  - ``` <a href=""><img src=""></a> ```
