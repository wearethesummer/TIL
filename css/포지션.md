## position 속성

  - Relative : 부모요소
  - Absolute : 자식요소

~~~
<div class="parent">
  <div class="child></div>
</div>

* 계층구조는 당연히 parent와 child으로 잡히지만, 부모요소에 position: relative를 선언하지 않으면
  child가 body 태그를 부모로 생각하고 브라우저를 기준으로 위치를 잡게된다. 
~~~

~~~
.parent {
  position: reletive;
}

.child {
  position: absolute;
  right: 0;
  bottom: 0;
}
~~~

> left, right, top,  bottom 속성은 반드시 포지션 속성과 같이 써야 작동된다.
