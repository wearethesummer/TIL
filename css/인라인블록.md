### 인라인요소
 - 한 줄에 여러개 배치
 - 기본 너비 값은 컨텐츠의 너비 값
 - 크기 값을 가질 수 없음
 - 상하 마진 가질 수 없음
 - ```span``` ```a``` ```small``` ```big``` ```em``` ```u``` ```s``` ```del``` ```br``` ```q``` ```b```
   ```strong``` ```mark``` ```sub``` ```sup``` ```video``` ```audio```

### 블록 요소
 - 한 줄에 한 개만 배치
 - 기 본 너비 값은 100%
 - 크기 값을 가질 수 있음
 - 상하좌우 마진 가질 수 없음
 - ```div``` ```table``` ```figure``` ```figcaption``` ```caption``` ```header``` ```nav``` ```footer```
   ```section``` ```article``` ```aside```
   <br/>
   ```p``` ```blockquote``` ```ul``` ```ol``` ```li``` ```td``` ```th``` ```form``` ```hr``` ```h1~h6```
